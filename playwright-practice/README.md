# Playwright Risk Form E2E Practice

이 폴더는 `AI Decision Risk Signal Monitoring System`의 핵심 개념을 단순한 웹 입력 폼으로 축소하여, Playwright 기반 E2E 테스트 흐름을 실습하기 위한 미니 프로젝트입니다.

현재 메인 프로젝트는 Python과 pytest 기반의 rule-based decision pipeline을 다룹니다.  
이 미니 실습은 디지털이엔에스 현장실습 직무와 연결하기 위해, 웹 화면에서 사용자 입력 → 버튼 클릭 → 결과 검증 흐름을 Playwright로 테스트하는 것을 목표로 합니다.

---

## Practice Goal

```text
1. Playwright 기본 테스트 구조 이해
2. 웹 입력 폼에 값 입력
3. 버튼 클릭 이벤트 실행
4. 화면에 출력된 결과 검증
5. E2E 테스트 흐름 경험
